// ===============================
// Application User Roles & Responsibilities
// ===============================

// 1. Super Admin
// - Full control over system configuration and user management
// - Manage global platform settings, permissions, and integrations
// - Oversee tenant onboarding for multi-tenant SaaS

// 2. Individual Property Owner (Self-Managing)
// - Register and manage personal rental properties
// - Oversee tenants, leases, rent collection, and maintenance directly
// - Access financial and performance reporting for owned units

// 3. Property Owner/Investor (With Management Company)
// - Monitor performance of properties managed by a company
// - Receive portfolio analytics, financial statements, and strategic insights
// - Limited direct involvement in daily operations

// 4. Real Estate Investor
// - Track and analyze multiple property investments
// - Access consolidated dashboards for performance metrics
// - Receive investment reports and recommendations

// 5. CEO
// - Oversee company-wide operations and key metrics
// - Executive-level reporting, portfolio summaries, and strategic dashboards
// - Manage user access and high-level settings

// 6. CFO
// - Monitor and analyze financial data across all properties
// - Generate financial statements, audit reports, and compliance documents
// - Oversee budgeting, cash flow, and risk management

// 7. Portfolio Manager
// - Manage and optimize groups of properties or regions
// - Monitor KPIs, occupancy rates, and benchmark analytics
// - Generate performance and comparison reports

// 8. Property Manager
// - Handle daily operations for assigned properties or portfolios
// - Manage tenants, leases, rent collection, communication, and maintenance requests
// - Oversee move-in/move-out processes, inspections, and renewals

// 9. Property Administrators
// - Support property managers with administrative tasks
// - Maintain property records, update unit information, and process documents
// - Coordinate lease paperwork and compliance tracking

// 10. Maintenance Coordinator
// - Receive, prioritize, and assign maintenance work orders
// - Schedule tasks for technicians and vendors, monitor completion
// - Manage inventory, vendor performance, and maintenance reports

// 11. Maintenance Technician
// - Receive and complete assigned work orders (maintenance/repairs)
// - Upload documentation, photos, and status updates from the field
// - Communicate with coordinators and property managers

// 12. Maintenance Vendor/Contractor
// - Access and complete assigned third-party maintenance tasks
// - Submit invoices/documentation and manage schedules
// - Coordinate with property staff for access and updates

// 13. Financial & Accounting Teams (Internal)
// - Process financial transactions, payments, and reconciliations
// - Maintain ledgers, tax documentation, and compliance
// - Liaise with property managers and executive team on reports

// 14. Tenant (Residential)
// - Access self-service portal for rent payments, maintenance, lease documents, and communication
// - Submit maintenance requests, track status, and review account history
// - Receive announcements and updates

// 15. Tenant (Commercial)
// - Similar to Residential Tenant but with business-focused features such as CAM reconciliation, larger lease terms, and multiple users per account
// - Access to payment processing, maintenance, and communication portals
// - Review documents, schedules, and account information
