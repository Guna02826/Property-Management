Properties:

Buildings
- Create buildings with name, address, coordinates, and total floors

Floors

- Set floor-level amenities

Spaces

- View space-level occupancy heatmaps

- Compare multiple spaces side-by-side
- View space details (name, size, price, location, status, amenities, images)
- Exclude non-leasable spaces from client search results
- Validate gross_sqft >= usable_sqft
- Support usage types (Office, Canteen, Restroom, Storage, Corridor, Janitor, Other)
